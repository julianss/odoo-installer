About

The Odoo Management Dashboard provides a web interface for managing your Odoo Docker environments. Features include:

  • Container status and control
  • Real-time log streaming
  • Git repository management
  • Backup creation and scheduling
  • Database copy between environments

Documentation and source code available at the project repository.
